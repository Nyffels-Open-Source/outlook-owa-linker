Outlook OWA Linker

Outlook add-in om OWA-deeplinks te kopiëren en items te openen op ID. Werkt in New Outlook (Mac/Win) & Outlook on the web.

Installatie (via bestand)

  1. Klik hieronder op Download manifest.
  2. Open Outlook on the web: outlook.office.com
  3. Ga naar Get Add-ins → My add-ins → Custom add-ins → Add from file.
  4. Selecteer het gedownloade manifest.xml bestand.
  5. Klik op Install en sluit het venster.

⬇️ Download manifest.xml

Na installatie verschijnt de add-in ook in de desktop-app (zelfde account).
Broncode: GitHub repo

Gebruik

  • Copy OWA Link (ribbon): kopieert een stabiele OWA-deeplink van het huidige bericht.
  • Open by ID (taskpane): plak een ItemId of volledige OWA-link → opent in Outlook; zo niet, valt terug op OWA (browser).

Opmerkingen

  • OWA-links openen altijd in de browser (by design). De client-open gebruikt displayMessageForm met fallback.
  • Outlook cached add-ins. Bij updates: verhoog <Version> in manifest.xml om een refresh te forceren.
