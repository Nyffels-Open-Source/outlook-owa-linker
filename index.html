Outlook OWA Linker

Outlook add-in to copy OWA links and open items by ID. Works in New Outlook (Mac/Win) & Outlook on the web.

Install (sideload)

  1. Open Outlook (web or new desktop) → Get Add-ins → My add-ins → Add from URL.
  2. Paste the manifest URL below → Install.

Manifest URL

https://nyffels-open-source.github.io/outlook-owa-linker/manifest.xml

Direct files

  • manifest.xml
  • functions.html
  • taskpane.html

Usage

  • Copy OWA Link (ribbon): copies a stable OWA deeplink of the current message.
  • Open by ID (taskpane): paste an ItemId or full OWA link → opens in Outlook; if unsupported, falls back to OWA in a new tab.

Notes

  • OWA links always open in the browser (by design). Opening in the client uses displayMessageForm with graceful fallback.
  • Outlook caches add-ins. If you update, bump <Version> in manifest.xml to force refresh.
