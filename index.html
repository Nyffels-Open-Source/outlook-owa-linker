Outlook OWA Linker

An Outlook add-in to copy OWA deeplinks and open items by ID. Works in New Outlook (Mac/Win) & Outlook on the web.

Installation (via file)

  1. Click Download manifest below.
  2. Open Outlook on the web: outlook.office.com
  3. Go to Get Add-ins → My add-ins → Custom add-ins → Add from file.
  4. Select the downloaded manifest.xml file.
  5. Click Install and close the window.

⬇️ Download manifest.xml

After installation, the add-in also appears in the desktop app (same account).
Source code: GitHub repo

Usage

  • Copy OWA Link (ribbon): copies a stable OWA deeplink of the current message.
  • Open by ID (taskpane): paste an ItemId or full OWA link → opens in Outlook; if not supported, falls back to OWA in a browser tab.

Notes

  • OWA links always open in the browser (by design). The client-open uses displayMessageForm with fallback.
  • Outlook caches add-ins. When updating, increase the <Version> in manifest.xml to force a refresh.
